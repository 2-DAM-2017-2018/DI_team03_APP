Proyecto de solicitud de recursos
=================================

Descripción del proyecto
------------------------
Esta aplicación está pensada para institutos y coleguios con necesidad
de usar un sistema de solicitud de recursos (patio, gimnasio, salón de
usos múltimles, etc). Con la aplicación puedes solicitar un recurso 
para una fecha concreta y una hora, o ver si ya está solicitado.
